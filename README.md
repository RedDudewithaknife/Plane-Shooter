# Plane-Shooter
Plane shooter is a 2D game, where a player has to shoot or dodge explosives. The program was made on [https://www.mblock.cc/en/]  
This game only supports PC as it is keybind dependant
Keybinds W to go up
         S to go down
         Space to shoot
